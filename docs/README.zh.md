## 推荐阅读顺序

1. 先读 [当前产品总览](./current-product-overview.zh.md)，理解现在真实落地的产品形态。
2. 再读 [多 Agent 架构](./multi-agent-architecture.zh.md)，理解系统分层。
3. 再读 [统一 Agent 协议 v1](./unified-agent-protocol.zh.md)，理解共享运行时契约。

5. 新增 provider 时，再读 [Agent 扩展指南（中文）](./agent-extension-guide.zh.md)。
